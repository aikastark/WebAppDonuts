## Technologies Used

- Node.js
- Express.js
- HTML
- CSS
- JavaScript

## Features

- Browse menu items
- Add items to cart
- Order management
- Payment section

## Project Structure

controllers – application logic  
views – frontend templates  
routes.js – application routes  
dbConnection.js – database connection  

## Installation

1. Clone repository
2. Run

npm install

3. Start server

node index.js
